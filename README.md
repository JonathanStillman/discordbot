## How To Use

To clone and run this application, you'll need [Git](https://git-scm.com) and [Node.js](https://nodejs.org/en/download/) (which comes with [npm](http://npmjs.com)) installed on your computer. YOU MUST HAVE AN ACCOUNT / SERVER FROM DISCORD TO ACTIVATE BOT. From your command line:

```bash
# Clone this repository
$ git clone https://github.com/JonathanStillman/crypto-currency-app.git

# Go into the repository
$ cd discordbot

# Install dependencies
$ npm install

# Run the app
$ npm run test
```

> **Note**
> If you're using Linux Bash for Windows, [see this guide](https://www.howtogeek.com/261575/how-to-run-graphical-linux-desktop-applications-from-windows-10s-bash-shell/) or use `node` from the command prompt.
